## HARW DAQ system software

Software to control the HARW DAQ system

#### ACAPS emulator
Within ![src_python](/src_python/) is the code to run an ACAPS emulator via a streamlit python app

#### HARW DAQ system control
Within ![src_matlab](/src_matlab/) is the code to create two ACAPS clients
- HARW client: client to control the NI DAQ
- Imetrum Client: client to control the Imetrum camera system
