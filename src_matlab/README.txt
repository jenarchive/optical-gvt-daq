Codebase to control the HARW DAQ system and camera system.

"Channels.csv" is a list of channels used durign testing - this csv file is used to setup the channels automatically

"DummyConfig.nce" is a config file for NImax to emulate the NI DAQ on any PC

"setup_HARW.m" and "Setup_CAM.m" were sued to setup the two ACAPS clients

"example_chirp.m" gives an example generated chirp showing all the options
